A selection of my open-source projects on [GitHub](https://github.com/ericfitz), grouped by topic. Almost everything is Apache 2.0 and/or MIT licensed.

## TMI - Threat Modeling Improved - Threat modeling platform for humans and agents

[TMI Wiki](https://github.com/ericfitz/tmi/wiki)

- [tmi-ux](https://github.com/ericfitz/tmi-ux) — Web front-end for threat-model and data-flow-diagram editing, with OAuth/OIDC and real-time collaborative editing.
- [tmi](https://github.com/ericfitz/tmi) — REST API server and platform for humans and agents to run security reviews and threat modeling.
- [tmi-tf-wh](https://github.com/ericfitz/tmi-tf-wh) — Webhook-driven, cloud-hosted Terraform analyzer for TMI.
- [tmi-clients](https://github.com/ericfitz/tmi-clients) — REST clients for the TMI threat-modeling API.

## Security tools

- [sqdist](https://github.com/ericfitz/sqdist) — Rust CLI that scores how likely one string is spoofing or squatting on another.
- [pypi-proxy](https://github.com/ericfitz/pypi-proxy) — Experimental local PyPI pull-through cache with hooks to support security scanning before use.
- [asciicast-utils-python](https://github.com/ericfitz/asciicast-utils-python) — Python utilities to record and play back terminal sessions as asciicast files.
- [dominfo](https://github.com/ericfitz/dominfo) — Gathers domain-name info to assess subdomain-takeover risk.

## Cloud, DNS & infrastructure

- [r53](https://github.com/ericfitz/r53) — Python CLI for AWS Route 53: simple record management and dynamic DNS.

## Hardware projects

- [lcars-pi-clock](https://github.com/ericfitz/lcars-pi-clock) — LCARS (Star Trek: TNG) themed clock/calendar for a WaveShare e-ink display on Raspberry Pi.

## File & document utilities

- [scan-namer](https://github.com/ericfitz/scan-namer) — Renames scanned documents in Google Drive with descriptive names based on their content.
- [renamer](https://github.com/ericfitz/renamer) — Local file renamer.

## Web & UI experiments

- [css-animations](https://github.com/ericfitz/css-animations) — Exploring CSS animations to explain user interaction in lieu of written help.
- [gesture-test](https://github.com/ericfitz/gesture-test) — Experiments toward an asynchronous touch-gesture library for Python.
